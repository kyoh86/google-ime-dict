## gimedic activate

Activate previously scheduled sync configuration

```
gimedic activate [flags]
```

### Options

```
      --exec string          Executable path for scheduled jobs (optional)
  -h, --help                 help for activate
      --interval duration    Sync interval (default 5m0s)
      --journal-dir string   Shared journal directory (optional)
      --path string          Local user_dictionary.db path (optional)
```
